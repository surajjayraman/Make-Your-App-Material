Project Overview:
In this project, an app is redesigned to follow the Material Design guidelines and translate a set of static design mocks to a living and breathing app. It also replicates a common developer task of updating and changing an app's design as new standards are released.

Project Goals:
Understand the fundamentals of Android design.
Apply Material Design guidelines to a mobile application.
Separate an interface into surfaces.
Effectively use transitions and motion.

Project Tasks:
Improve the XYZ Reader app: A mock RSS feed reader featuring banner photos and shocking headlines! 
Take the user feedback in the UI Review node and implement changes that will improve the UI and make it conform to Material Design.
